# 🔐 Secure File Encryption & Decryption System

A Python-based application that securely encrypts and decrypts files using password-based encryption.  
The project includes both **Command-Line (CLI)** and **Graphical User Interface (GUI)** versions.

---

## 📌 Project Overview

Data security is a critical requirement in modern IT systems.  
This project demonstrates how sensitive files can be protected from unauthorized access using encryption techniques.

The application allows users to:
- Encrypt files securely using a password
- Decrypt encrypted files using the same password
- Protect confidential data from misuse or theft

---

## ✨ Features

✔ Password-based file encryption  
✔ Secure file decryption  
✔ Industry-standard cryptography (Fernet)  
✔ Command-Line Interface (CLI) version  
✔ Graphical User Interface (GUI) using Tkinter  
✔ Error handling for wrong passwords or corrupted files  

---

## 🛠 Technologies Used

- **Python**
- **Cryptography (Fernet)**
- **Tkinter (GUI)**
- **File Handling**
- **Hashing (SHA-256)**
